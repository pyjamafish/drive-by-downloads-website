Androzek

Name Androzek
Consequences Adds browser extensions, modifies a specific DDL per target browser, and changes browser settings to insert additional, unauthorized ads into web pages, often on top of legitimate ads from search engines
Date May 2020 - Now
(peak in August 2020)

Where did it come from?

Unknown. Designed to modify a browser’s Dynamic Link Libraries to change the settings, resulting in popup ads.


How was it caught?

Microsoft has recorded thousands of encounters in windows PCs.


Who/what does it target?

PCs all around the world. Heavy concentrations of attacks in Europe, South Asia and Southeast Asia.


How does it work?

The malware is distributed via classic drive-by download schemes. Attackers use sprawling infrastructure
to distribute hundreds of thousands of unique Androzek installer samples, each follows the structure of
the image above. The installer drops and .exe file with a random name, and makes it look legit. The
malware is then installed like a usual program.


What damage is it capable of doing?

Intended effects for users to inadvertently click on these malware-inserted ads, which lead to affiliated pages.
The attackers earn through affiliate advertising programs, which pay by the amount of traffic referred to sponsored
affiliated pages. The real threat is that the malware can steal login credentials from the Firefox browser.


Has a fix been found? How can we defend ourselves?

There is no fix yet. Andozek could potentially be used for more malicious crimes. It hasn’t been implemented in all
websites yet, but this malware continues to spread. We can defend ourselves with Microsoft defender antivirus.
